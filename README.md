# 🧭 Travel Destination Landing Page

This is a simple, responsive travel landing page built with just HTML and CSS. It highlights a beautiful travel destination and can be easily customized for any country, city, or attraction.

Whether you're promoting a location, creating a mockup for a client, or just experimenting with front-end design — this project is a great starting point.

---

## 🌟 What It Includes

- A clean, scroll-friendly layout
- A featured image and intro section
- Destination details (climate, language, currency)
- A call-to-action button (e.g., “Plan Your Trip”)
- Pure HTML & CSS — no frameworks or dependencies
